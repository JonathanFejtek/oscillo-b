What is Oscillo?

Oscillo is a prototype generative design tool that uses harmonic mathematics to create cool forms. Generative design can be broadly defined as a modern design movement that emphasizes the creation of algorithmic and mathematical systems to aid in various design problems. Oscillo is a generative form-finder that can produce an array of forms given different input parameters.

Oscillo is a also a technical web-development experiment. Oscillo is built with a custom component system written almost entirely in VanillaJS. Oscillo does not depend on any Javascript libraries (besides some very minimal jQuery). If you are interested, you can view the source here.

How does it work?

Oscillo's core functionality is based on what is called a Fourier Series in mathematics. Put simply, Oscillo relies heavily on the principle that any mathematical function can be represented as a sum of simple waves. In other words, we can create mathematical functions by adding simple waves together. In this sense, one might understand Oscillo as a visual additive synthesizer.

Oscillo allows us to modify two distinct spectra to produce two different mathematical functions. Firstly, Oscillo allows us to modify a spectrum that is a sum of sine waves. And secondly, Oscillo allows use to modify a spectrum that is a sum of cosine waves.
